# 情報の抽出

**発信者（発）:** 不明  
**受信者（着）:** 不明  
**日時:** 1966年1月19日  

1. 1966年1月19日、セリオ・サンチェス・スアレスがフロリダ州マイアミで双発機の修理を行っていた。  
2. MIRR（革命的回復運動）に使用される予定で、北ラス・ビルタス州に人員を浸透させる計画があった。  
3. スアレスは、オーランド・ボッシュ・アビラがミーティングを行う予定であると述べた。  
4. ボッシュは、ミーティングのために資金を集める計画があった。  
5. ボッシュは、アメリカ合衆国の政府に対する抗議活動を計画していた。
